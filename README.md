# CLI Vector Editor

Простой векторный редактор с интерфейсом командной строки (CLI).

## Постановка задачи

Тестовое задание (этап 1):

> Создать простой векторный редактор с интерфейсом командной строки (CLI).
>
> **Что нужно сделать:**
> - Реализовать базовый функционал работы с фигурами: **точка**, **отрезок**, **круг**, **квадрат**
> - Добавить команды для: **создания фигур**, **удаления фигур**, **просмотра списка созданных фигур** (вывод в консоль)
> - Оформить код в Git-репозитории и предоставить ссылку

## Возможности

- Фигуры: точка, отрезок, круг, квадрат
- Команды: создание фигур, удаление по id, вывод списка фигур в консоль

## Установка

```bash
pip install -r requirements.txt
```

## Запуск

Из корня репозитория:

```bash
python -m src.main
```

## Команды

- `add point <x> <y>` — добавить точку
- `add segment <x1> <y1> <x2> <y2>` — добавить отрезок
- `add circle <x> <y> <r>` — добавить круг (центр и радиус)
- `add square <x> <y> <side>` — добавить квадрат (левый нижний угол и сторона)
- `list` — вывести список всех фигур с id
- `delete <id>` — удалить фигуру по id
- `quit` (или `exit`, `q`) — выход

## Пример

```
> add point 1 2
Added Point(1.0, 2.0) with id 1
> add circle 0 0 5
Added Circle(0.0, 0.0, 5.0) with id 2
> list
  1: Point(1.0, 2.0)
  2: Circle(0.0, 0.0, 5.0)
> delete 1
Deleted id 1
> quit
Bye.
```
