# CLI Vector Editor

Простой векторный редактор с интерфейсом командной строки (CLI).

## Постановка задачи

Тестовое задание (этап 1 и этап 2):

> **Этап 1.** Создать простой векторный редактор с интерфейсом командной строки (CLI).
> - Реализовать базовый функционал работы с фигурами: **точка**, **отрезок**, **круг**, **квадрат**
> - Добавить команды для: **создания фигур**, **удаления фигур**, **просмотра списка созданных фигур** (вывод в консоль)
> - Оформить код в Git-репозитории и предоставить ссылку
>
> **Этап 2.**
> - Добавить возможность **сохранения и загрузки** списка фигур в файл
> - Добавить поддержку новых фигур: **овал**, **прямоугольник**
> - Поддерживать удобный и понятный интерфейс командной строки

## Возможности

- **Фигуры:** точка, отрезок, круг, квадрат, овал, прямоугольник
- **Команды:** создание фигур, удаление по id, список фигур, сохранение и загрузка из файла (JSON)

## Установка

```bash
pip install -r requirements.txt
```

## Запуск

Из корня репозитория:

```bash
python -m src.main
```

## Команды

- `add point <x> <y>` — точка
- `add segment <x1> <y1> <x2> <y2>` — отрезок
- `add circle <x> <y> <r>` — круг (центр и радиус)
- `add square <x> <y> <side>` — квадрат (угол и сторона)
- `add oval <x> <y> <a> <b>` — овал (центр и две полуоси)
- `add rectangle <x> <y> <width> <height>` — прямоугольник (угол, ширина, высота)
- `list` — список всех фигур с id
- `delete <id>` — удалить фигуру по id
- `save <файл>` — сохранить список фигур в JSON
- `load <файл>` — загрузить список фигур из JSON
- `quit` (или `exit`, `q`) — выход

## Пример

```
> add point 1 2
Добавлено: Point(1.0, 2.0), id=1
> add oval 0 0 5 3
Добавлено: Oval(0.0, 0.0, 5.0, 3.0), id=2
> save shapes.json
Сохранено в shapes.json
> list
  1: Point(1.0, 2.0)
  2: Oval(0.0, 0.0, 5.0, 3.0)
> load shapes.json
Загружено из shapes.json
> quit
До свидания.
```
